#######################
Tanzen will ich, tanzen
#######################

*English translation "Dance Will I, Dance".*

:Written by: Barbara Büchner
:Illustraited by: Monika Liamgruber
:Published: 1994
:ISBN: 3-85264-449-6

Translation
===========

Following 1800-1900's English, British text (Grimms fairly tails style).
Using children's vocabulary but avoiding to overly simplify/dumb-down language.

After this initial pass and some manual improvements, I found the language read a little too awkwardly,
then I targeted 1920's British English, but selectively updated phrases, leaving some as-is,
where it read well and added some character (and in some cases a little gravitas) to the story.


Notes
=====

The Rat in the original story is not explicitly male/female, the German word for Rat is feminine
so to a child's ear it would sound feminine - as the imagery doesn't denote a gender this is left as female,
but it could be changed.

#############################
Translated Children's Stories
#############################

This repository contains English translations for children's picture books.

The intent is that each story is printed in a compact portrait A4 format,
which can be stored with the book and read aloud along side the book.


Building
========

The RST files may be compiled into PDF files.

Requirements
------------

- Sphinx
- LaTeX

Run::

  make

PDF files will be generated in the ``build/`` directory.
